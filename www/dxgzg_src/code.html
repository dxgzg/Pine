Pine网络库设计模型

  • 采用Rector模型,主流网络库采用的模型
  • 设计思想来源muduo网络库和libevent网络库
  • Http类根据《Linux高性能服务器编程》这一书的有限状态机思想写出来的
    one loop per thread模型，陈硕老师永远滴神！

Pine网络库使用方法如下

  1. 运行环境Linux系统,g++版本至少4.9以上,支持C++14版本,且装有cmake。
  2. 使用该网络库，配置好InetAddr的端口号和IP地址即可,使用Http的话,把Http的页面根目录配置好即可
  3. 已经写好了运行脚本,已有了运行环境,在命令终端输入./autobuild即可运行
